SkinCare App

(ENG) A prototype Android application for tracking personal skin care and building a trackable routine.

**The main goal of this repository** is to provide a practical base for practicing **Manual QA Engineer** skills. The project is used to study the structure of mobile applications, create test documentation, find bugs, and understand the software development life cycle.

Main functionality (Test object):
* Creating a skin care schedule (morning/evening routine).
* Controlling the frequency of use of active ingredients (retinol, salicylic acid, spikules, etc.) to avoid conflicts between products.
* Logging the history of product use.

Technical stack (for reference)
* **Platform:** Android
* **Language:** Kotlin
* **Interface:** Jetpack Compose
* **Build:** Gradle

This project was developed using artificial intelligence to generate the basic architecture and code.
